## The One-Sentence Autonomy Rule

**If an agent can take an action you would need to explain to Security, Legal, or a customer, it may propose â€” but it may not execute.**

---

## Non-Negotiable Kill Switches

Autonomous execution is prohibited if ANY of the following is true:

- The agent can deploy, delete, send, or merge without approval  
- The agent touches credentials, identity, or secrets  
- The agent processes untrusted external input  
- The agent can affect customer-visible or financial state  
- Actions are irreversible or cannot be reconstructed post-incident
